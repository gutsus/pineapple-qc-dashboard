# Keelings Pineapple QC Dashboard

## Quickstart (local)
1) Install Python 3.10+  
2) In this folder, install deps:
   pip install -r requirements.txt
3) Run the app:
   streamlit run streamlit_nav_app3.py

## Files
- streamlit_nav_app3.py     # main app
- mock_prototype_data_with_qc_defects.csv
- keelings_logo.png

## Notes
- Keep the CSV and logo in the same folder as the app script.
- If you later move to real data sources, we can load from Drive/S3/SharePoint.
